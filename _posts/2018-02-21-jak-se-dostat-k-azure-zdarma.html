Azure je skvělou platformou pro vaše inovace, aplikace, data, staré i nové IT. Řekněte svému zaměstnavateli jak skvělý je a prosaďte ho u vás, nechte si Azure koupit! Pokud ale pro váš osobní rozvoj, soukromý projektík, startup vašich dětí či neziskovou organizaci pro kterou děláte dobrovolníka potřebujete Azure, možná se dostanete k Azure úplně zdarma. Jaké máte možnosti?

Azure Free Account a Trial

Asi nejdůležitější je Azure Free Account. Ten je relativně nový a nahrazuje všechny předchozí programy typu trial apod. Jak to funguje? Musíte si zaregistrovat nový účet a ověřit, že nejste robot - k tomu potřebujete platné telefonní číslo a platební kartu (z té vám nic odečteno nebude, nemusíte se bát - objeví se rezervace, tedy nikoli odečet, na částku 0,844 EUR, která po pár dnech bude zase uvolněna). Pak už můžete využívat služeb Azure.

Prvních 30 dní - 170 EUR na cokoli

Na první měsíc dostáváte kredit 170 EUR, s kterým můžete dělat takřka cokoli. V trial subscription mohou být omezení na některé skutečně high-end typy VM (například M řada třeba s 128 core + 4TB RAM), ale jinak jsou všechny služby plně dostupné. IaaS, PaaS. VMka, databáze, aplikace, zpracování dat - zkrátka cokoli. Jen nezapoměňte, že z kreditu nemůžete platit za licence třetích stran v marketplace (tedy můžete mít VM s Windows, Ubuntu nebo CentOS, openSUSE, ale nepořídíte si RHEL, SUSE, F5 či Fortinet v pay-as-you-go modelu, maximálně v trial nebo se svou vlastní licencí).

Prvních 12 měsíců - využíváte vyjmenované služby zdarma

Od druhého měsíce nemáte žádný volný kredit, ale můžete využívat služby dostupné zdarma. Kromě těch v běžném katalogu (ty jsou navždy zdarma) nabízí měsíc 2-12 o dost víc. Tak například můžete mít i 750 hodin provozu VM typu B1S pro jednu Windows a jednu Linux VM, 128 GB SSD disků, 5 GB Blob Storage, 5 GB File Storage, malou Azure SQL (S0, 250 GB storage), malou Cosmos DB (400 RU) a zvýšený limit odchozího síťového provozu (15 GB měsíčně). Toto jsou služby, které normálně zdarma nejsou.

Kromě toho můžete využívat služby, které jsou zdarma trvale a pro kohokoli (viz dále).

Po 12 měsících pokračujete 25+ službami, které máte navždy zdarma

Uběhl vám rok rychle? Nevadí, i tak můžete svůj účet nadále používat s využitím služeb, které jsou navždy zdarma. Patří sem například 10 webových aplikací, velmi slušný příděl spuštění Azure Functions (pokud šetříte pamětí a časem dostanete se na 1 000 000 spuštění měsíčně), 50 000 objektů v Azure Active Directory a současně v AAD B2C, 5 vývojářů ve Visual Studio Team Services, monitoring aplikací, Machine Learning (100 modulů, 1 hodina na experiment), IoT Hub (8000 zpráv denně), Azure Search, milion push notifikací s Notification Hub, základní možnosti Data Factory a Data Catalog, 5OO minut Azure Automation, 5 GB odchozího provozu a celá řada kognitivních služeb se základním přídělem dotazů za měsíc (rozpoznávání obličejů, řeč, překlady).

Registrace je tady: https://azure.microsoft.com/en-us/free/

Podrobnosti zde: https://azure.microsoft.com/en-us/free/free-account-faq/

Azure pro vývojáře v rámci Visual Studio

Následující varianta není vyloženě zdarma, pakliže jste už neinvestovali do vývojových platforem Microsoftu v různých balíčcích Visual Studio (dříve také MSDN). Tak například pokud se rozhodnete pro Visual Studio Professional s platbou 539 USD za rok, dostanete Visual Studio IDE, základní funkce VSTS, různá školení a jiné vzdělávací programy, slevu na licence a zdroje v Azure (např. neplatíte za licence SQL za určitých podmínek) a kromě toho máte 50 USD kredit v Azure.

Pro Visual Studio Enterprise dostanete 150 USD kreditů měsíčně a pro klasické MSDN je to 100 USD.Azure pro Proof of Concept

Pokud máte obchodní vztah s Microsoftem a toužíte vyzkoušet, jestli Azure opravdu dokáže splnit vaše požadavky a podmínky Free Accountu na to nestačí, obraťte se na své kontakty u Microsoftu. Existuje pár programů, které umožňují obchodníkům a Azure specialistům získat prostředky na společné PoC.

Azure pro startupy

Děláte na něčem opravdu zajímavém? Jste malý startup a výbornými nápady a potřebujete pro svoje řešení cloud? Microsoft nabízí pro tyto případy už dlouhou dobu různé programy (nejznámější byl Bizspark) a nově je sjednotil do Microsoft for Startups: https://startups.microsoft.com/en-us/program-details/

Na stránce se dočtete jak to funguje, v zásadě Microsoft si s vámi popovídá, jsou tam různá kritéria a zjednodušeně řečeno se zjistí, zda váš startup a Microsoft spolu mohou změnit svět :)  Pokud ano, můžete se dostat až ke 120 000 USD v Azure kreditech, získat přístup k vývojovému portfoliu Visual Studio, zvednout svou produktivitu s Office365 a rozjet správu svých nových zákazníků s Dynamics365. Kromě toho může dosáhnout i na obchodní incentivy, například zařízení do katalogu pro co-sell (Microsoft obchodníci budou motivovaní pomoci vám vaše řešení nad Azure prodávat), mentoring, účast na konferencích apod.

Azure pro neziskové organizace

Pokud je vaší vášní pomáhat ostatním přes neziskovou organizaci, tak pokud splníte podmínky programu, můžete se dostat do programu Microsoft pro neziskovky (více zde: https://www.microsoft.com/en-us/nonprofits). Konkrétní podmínky pro získání statutu najdete tady: https://www.microsoft.com/en-us/nonprofits/eligibility

Pakliže podmínky splňujete můžete se kromě jiných Microsoft řešení jako je Office365 dostat i k 5 000 USD měsíčně v Azure.

 

Chcete si Azure jen vyzkoušet, využít pro svůj domácí IoT projekt, učit svoje děti programovat, začít přenášet svoje programátorské zkušenosti z Visual Studio třeba do oblasti serverless, ubezpečit se, že vaše firemní záměry se s Azure dají realizovat, zajistit rozvoj vašeho nadějného startupu nebo využít Azure pro bohulibé účely vaší neziskové organizace? Prozkoumejte možnosti získání Azure zdarma.
