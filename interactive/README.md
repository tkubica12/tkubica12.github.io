# Interactive articles

Experimental source and assets for future `/new/` articles.

Current rules:

- Canonical sources live in `interactive\source\YYYY\slug.article.md`.
- Shared generation assets live with the generator skill in `.agents\skills\interactive-article-generator\assets\`.
- Generated public output is written after Jekyll into `_site\new\YYYY\slug\`.
- Generated HTML is not the source of truth; edit the `.article.md` source or shared assets instead.

The first test article is `interactive\source\2026\token-saving-cz.article.md`.
