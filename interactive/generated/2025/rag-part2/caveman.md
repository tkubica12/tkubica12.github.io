# Vlastní data (RAG) pro vašeho AI agenta část druhá - graf znalostí a pokročilé metody

Knowledge graph, sumarizace konceptů a průchod grafem bez frameworkové magie. Pořád Python, PostgreSQL a filmová data.

## Key points
- Od dokumentů ke konceptům
- Inspirace GraphRAG, ale pořád bez magie
- Jak se na graf ptát
- Notebooky a celý postup
- Extrakce konceptů
- Uložení grafu do PostgreSQL
- Sumarizace konceptů
- Vektorizace mimo grafové properties
- Sada otázek pro porovnání
- Čtyři strategie vedle sebe
- Co z experimentu vychází
- Ručně zvolený plán pro otázku jako Star Wars
- Směr: agent plánuje vlastní průchod
