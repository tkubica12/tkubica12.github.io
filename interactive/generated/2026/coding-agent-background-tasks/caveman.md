# Coding agenti - paralelní práce agentů na vašich úkolech

Jak použít GitHub Copilot CLI, git worktree a background agenty ve VS Code, když chcete lokálně rozběhnout víc agentů bez toho, aby si navzájem přepisovali soubory.

## Key points
- GitHub Copilot CLI - agent bez IDE běžící prakticky kdekoli
- Manuální použití vícero agentů s Git worktree
- Agent HQ: založení background agenta
- Sledování, přepínání a handoff agentů
- Přijetí změn zpět do hlavního workspace
- Moje rozhodovací mapa: kdy použít co
